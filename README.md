## インストール方法

```bash
$ git clone git@github.com:saionji-bit/robosys2025.git
$ cd robosys2025
$ chmod +x closet

## closet コマンド

closet は，服のリストを「季節ごと」にグループ分けして表示するコマンドです。
自分のクローゼットにある服をテキストファイルで管理しておき，季節ごとの持ち物を確認する用途を想定しています。

## 入力フォーマット

1 行に 1 着分の情報を，カンマ区切りで次のように書きます。

季節,カテゴリ,色,名前

冬,コート,黒,ダウン
冬,ニット,白,セーター
夏,Tシャツ,白,UT
夏,パンツ,青,ジーンズ

closet では，次のようなルールで入力を解釈します。

先頭が # の行はコメントとして無視します。

空行は無視します。

「季節,カテゴリ,色,名前」の 4 要素になっていない行はエラーとして扱い，終了コード 1 で終了します。

## 使い方の例
ファイル名を引数で渡す場合

上の内容を clothes.txt という名前で保存しておきます。

```bash
$ ./closet clothes.txt
冬:
  コート 黒 ダウン
  ニット 白 セーター
夏:
  Tシャツ 白 UT
  パンツ 青 ジーンズ

標準入力から渡す場合
$ cat clothes.txt | ./closet
冬:
  コート 黒 ダウン
  ニット 白 セーター
夏:
  Tシャツ 白 UT
  パンツ 青 ジーンズ

## テスト

```bash

テスト内容は次のようになっています。

コメント行を含む入力
